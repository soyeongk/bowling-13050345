## 검증할 동작
한 프레임에서 스페어(2회 투구 합이 10)가 나오면, 그 프레임의 점수는 10 + 다음 1회 투구 핀 수여야 한다.

## 작성할 테스트
`test_game.py`에 다음 테스트를 추가한다. 1프레임에서 5,5(스페어) 후 3을 굴리고, 나머지는 모두 거터.
프레임1 = 10+3 = 13, 프레임2 = 3+0 = 3, 나머지 0 → 총 16점.

```python
def test_one_spare_adds_next_roll_as_bonus():
    game = Game()

    game.roll(5)
    game.roll(5)
    game.roll(3)
    for _ in range(17):
        game.roll(0)

    assert game.score() == 16
```

## 구현 계획 (테스트를 통과시키기 위한 프로덕션 코드)
`score()`를 프레임 단위 순회로 재작성한다 (기존 `sum(self._rolls)` 방식은 스페어 보너스를 표현할 수 없으므로 교체).

- 10개 프레임을 순회하며 인덱스 `i`로 롤 목록을 훑는다.
- 각 프레임은 기본적으로 2회 투구(`rolls[i]`, `rolls[i+1]`)를 소비한다.
- 두 투구의 합이 10(스페어)이면 다음 1회 투구(`rolls[i+2]`)를 보너스로 더한다.
- 프레임 점수를 총점에 누적하고 `i += 2`로 다음 프레임으로 이동한다.
- 스트라이크(1회 투구로 10) 처리는 아직 하지 않는다 — 이번 테스트 입력에는 스트라이크가 없다.

## 범위 제한 (이번 사이클에서 하지 않는 것)
- 스트라이크 보너스, 10번 프레임 규칙은 다루지 않는다.
